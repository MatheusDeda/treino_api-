# treino_api-
Treino_API
COMO EXECULTAR 
1. Clone o repositório e acesse a pasta do projeto: cd curso_api
2. 2. Instale as dependências necessárias:pip install "fastapi[all]"
pip install "pydantic[email]"
3. 3. Inicie o servidor local com Uvicorn:uvicorn main:app --reload
4. Acesse a documentação interativa (Swagger UI):
Abra o seu navegador e acesse: http://127.0.0.1:8000/docs.
Através do Swagger, você poderá testar todas as rotas de forma visual.
